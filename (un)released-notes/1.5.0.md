### New Features
- Implemented globally emulator events dispatch (to receive events emitted by the emulator in your OBS source for example);
  - All events dispatched by the emulator has a flag `unichat:emulator_generated` to identify them as emulator-generated events;
- Added warn messages to widgets;
- Added support for custom icons in scrapers (only `data:` URLs);

### Fixes and Improvements
- Plugin kick moved to [`unichat-community/unichat-plugin-kick`](https://codeberg.org/unichat-community/unichat-plugin-kick);
- Improved releases and third party licenses loading;
- Fixed gallery assets loading;
- Renamed widget files:
  | Before      | Now           |
  |-------------|---------------|
  | `style.css` | `widget.css`  |
  | `main.html` | `widget.html` |
  | `script.js` | `widget.js`   |

### Other Changes
- Changed license from MPL-2.0 to EPL-2.0;
- Fixed width and height of main window to `1024x576`;
- Moved docs to another repository;
- Fully refactored interface;
  - Removed mantine and all react stack, replaced by preact with tailwindcss and self implemented components.
